### Sloppy

@InProceedings{marx:19:sloppy,
	title={Identifiability of Cause and Effect using Regularized Regression},
	author={Marx, Alexander and Vreeken, Jilles},
	booktitle={Proceedings of the 25th ACM SIGKDD International Conference on Knowledge Discovery and Data Mining},
	year={2019},
	organization={ACM}
}

The structure of this project is as follows:

Sloppy.R  		--- contains the code for Sloppy
utilities.R 		--- contains helper functions for Sloppy
test_benchmark.R	--- executes the experiments from the paper
bench_utils.R		--- contains helper functions to run the experiments
baselines.R		--- contains the baseline methods

results			--- folder that contains the result files that are generated when executing tests_benchmark.R
data			--- contains the benchmark data sets
